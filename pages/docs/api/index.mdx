There are two types of API for the Green Analytics project. 

## Tool API

This API can be used to create new tools for usage with Green Analytics. This API includes endpoints `logEvent` and `setPerson`.

To learn more about this API consult the [Tool API page](/docs/api/tool)

## Data API

This API can be used to extract information from your team's data pool. This can be used to handle the data externally, this could for example be using tools such as Excel or PowerBI.

To learn more about this API consult the [Data API page](/docs/api/data)
